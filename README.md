# Feishu Docx To Markdown

可以将飞书文档，一键复制为 Markdown 格式的纯文本，便于粘贴使用。本脚本不面向大众，仅用于本人特定的工作场景，建议 Fork 后自行修改。

## 使用方法

1. 等待加载完毕，全选 (连续按下 Ctrl+A) 文章
2. 从头滚动至页面底端，建议反复滚动两次 (或许吧……)
3. 点击右上角的一键复制
4. 调整纯文本格式的 Markdown

## 支持的格式

- 二级标题
- 三级标题
- 表格
- 无序列表
- 图片,仅替换为![](https://)，需手动更改链接

## 已知问题

- 可能漏掉头尾内容
- 格式需要微调
- 仅支持部分格式
